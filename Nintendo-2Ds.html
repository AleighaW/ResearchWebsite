Nintendo 2DS

The Nintendo 2Ds is a system that can play all the original Ds and 3Ds games on. This console came out on October 12th, 2013, and can only put on a 2D display. No games are made for this console anymore since it is so old now. Moreover, this console isn't even sold in stores anymore.
